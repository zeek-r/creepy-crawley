# creepy-crawley

## Crawling Golang
---
### Pre-requisites
- golint
> go install  golang.org/x/lint/golint
- gocyclo
> go install github.com/fzipp/gocyclo/cmd/gocyclo@latest


## Crawling Node.js backend
---
### Pre-requisites
- eslint
> npm i eslint@latest
- prettier
> npm i prettier@latest
- nyc

### Make executable
> chmod +x creepy-go.sh

### Run
./creepy-[lang].sh
